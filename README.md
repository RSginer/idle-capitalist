### Features
  - ✅ Buy Business.
  - ✅ Upgrade Business.
  - ✅ Manage orders.
  - ✅ Hire Managers.
  - ✅ Managers automatically manage orders.
  - ✅ Your businesses make money for you while you are out, even if the server is down or the websocket is disconnected.
  - ✅ Websocket reconnection.
